# Public Runtime Assets

Files in this directory are served from the site root by Vite and by the standalone Node server.

- `icon.svg`: app icon used by HTML metadata, the manifest, and README branding.
- `site.webmanifest`: install metadata for browsers.
- `yang-boss.png`: runtime image asset for the Boss Yang experience.

Documentation-only artwork belongs in `docs/assets/` instead of this directory.
